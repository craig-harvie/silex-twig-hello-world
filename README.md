## silex-twig-hello-world

#### Installation

**Step 1** - Install node dependencies

```
npm install
```

**Step 2** - Install php dependencies

```
composer install
```

**Step 3** - Build sass

```
gulp build
```

**Step 4** - Run server

```
./script/server
```

**Step 5** - Visit http://localhost:8080 and Enjoy!
